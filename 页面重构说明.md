## 10. 页面跳转关系

### 从服务购买页面跳转
- **入口**：`components/order/purchase/ServicePurchaseFooter.vue` 中的底部导航按钮
- **跳转方法**：
  ```javascript
  // 跳转到客服中心
  handleCustomerService() {
    uni.navigateTo({
      url: '/pages/mine/customer-service'
    })
  }
  
  // 跳转到设计师店铺
  handleShop() {
    uni.navigateTo({
      url: '/pages/designer/detail'
    })
  }
  
  // 跳转到预约页面
  handleBooking() {
    uni.navigateTo({
      url: '/pages/designer/booking'
    })
  }
  ```

### 从主页面跳转
- **设计师入口**：`components/ProfileSection.vue` 中的"设计师"快捷操作按钮
- **优服务入口**：`components/ProfileSection.vue` 中的"优服务"快捷操作按钮
- **品牌馆入口**：`components/ProfileSection.vue` 中的"品牌馆"快捷操作按钮
- **设计师卡片入口**：`components/ProfileSection.vue` 中设计师板块的设计师卡片
- **跳转方法**：
  ```javascript
  handleQuickAction(action, index) {
    if (action.title === '设计师') {
      uni.navigateTo({
        url: '/pages/designer/index'
      })
    } else if (action.title === '优服务') {
      uni.navigateTo({
        url: '/pages/service/index'
      })
    } else if (action.title === '品牌馆') {
      uni.navigateTo({
        url: '/pages/brand/index'
      })
    }
  }
  
  handleDesignerClick(designer) {
    uni.navigateTo({
      url: `/pages/designer/detail?id=${designer.id || 1}&name=${encodeURIComponent(designer.name || '')}`
    })
  }
  ```

### 从设计师页面跳转
- **入口**：`pages/designer/index.vue` 中的标签切换
- **设计师卡片入口**：`components/DesignSection.vue` 中排行榜和附近推荐列表的设计师卡片
- **跳转方法**：
  ```javascript
  switchTab(value) {
    if (value === 'service') {
      uni.redirectTo({
        url: '/pages/service/index'
      })
    } else if (value === 'brand') {
      uni.redirectTo({
        url: '/pages/brand/index'
      })
    }
  }
  
  handleStylistClick(stylist) {
    uni.navigateTo({
      url: `/pages/designer/detail?id=${stylist.id || 1}&name=${encodeURIComponent(stylist.name || '')}`
    })
  }
  ```

### 从优服务页面跳转
- **入口**：`pages/service/index.vue` 中的标签切换
- **跳转方法**：
  ```javascript
  switchTab(index) {
    if (index === 0) {
      uni.redirectTo({
        url: '/pages/designer/index'
      })
    } else if (index === 2) {
      uni.redirectTo({
        url: '/pages/brand/index'
      })
    }
  }
  ```

### 从品牌馆页面跳转
- **入口**：`pages/brand/index.vue` 中的标签切换
- **跳转方法**：
  ```javascript
  switchTab(index) {
    if (index === 0) {
      uni.redirectTo({
        url: '/pages/designer/index'
      })
    } else if (index === 1) {
      uni.redirectTo({
  ```
