# WAW Manager - 美发管理系统(商家端)

## 项目概述

WAW Manager 是一款基于 uni-app 开发的美发店铺管理小程序,主要面向商家端使用。项目采用 Vue 3 Composition API + Pinia 状态管理。

## 技术栈

- 框架: uni-app (Vue 3)
- 状态管理: Pinia + pinia-plugin-persistedstate
- 样式: SCSS
- 构建工具: uni-app 内置

## 开发注意事项

1. 页面需要一比一还原设计图,包括:
   - 布局结构
   - 颜色配色
   - 字体大小
   - 间距圆角
   - 图标样式

2. 交互逻辑要完整:
   - 所有按钮都需要有对应的点击事件
   - 弹窗需要有确认/取消逻辑
   - 列表需要支持空状态显示
   - Tab切换需要平滑过渡

3. 页面跳转要完整:
   - 使用 uni.navigateTo 进行页面跳转
   - 需要传参时使用 URL query 或 storage
   - 返回时使用 uni.navigateBack

4. 数据状态管理:
   - 使用 Pinia 管理全局状态
   - 页面数据使用 ref/reactive
   - 缓存数据使用 uni.setStorageSync

5. 组件复用:
   - 弹窗组件可复用 BaseModal
   - 按钮样式可复用 BaseButton
   - 卡片布局可复用 BaseCard
   - 列表布局可复用 BaseList
